BaseTypeContractResolver

This contract resolver will force JSON.Net to serialize a given type as the closest registered base type. This resolver can be used to avoid serializing properties in classes derived from registered base types. All one needs to do is add a base type to the resolver or at static initialization time to BaseTypes.Set.
